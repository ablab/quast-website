SPAdes - IDBA collaboration

This page is created for collaboration between SPAdes and IDBA-UD genome assemblers developers.

Data set

Our test data set is E. coli K-12, strain MG1655, single cell MDA, lane 1. It is downloaded from UCSD website. Direct link to the data set is here (you should enter Username: single-cell and Password: SanDiegoCA).

The data set statistics:
  • # reads: 29,124,078
  • Read length: 2 x 100
  • Insert size μ: 266.8
  • Insert size σ: 56.4
Reference for this data set can be downloaded from NIH website. Click on "Send" near the top right; select "Complete Record"; destination "File"; format "FASTA"; and save it.
