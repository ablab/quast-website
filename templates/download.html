Download

QUAST is an open-source project, you can browse, download or contribute to the source code of both the web server and its command-line engine at GitHub.

For installing the command-line version of QUAST (works on Linux and macOS), we recommend using common package managers:

Conda: conda install -c bioconda quast
Homebrew (Brewsci/bio): brew install quast
PyPI: pip install quast

You may also download the latest and older QUAST releases from SourceForge (latest, all versions) or GitHub (latest, all versions). Installation is pretty straightforward and described in the manual.
