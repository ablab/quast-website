Web interface

The QUAST web interface was available at http://quast.bioinf.spbau.ru/ since 2012 and until the end of 2019. Unfortunately, this web address and underlying hardware infrastructure are no longer available. As of February 2020, we are looking for another stable location for relaunching the webserver. Please stay tuned for further notices here and on Twitter. Sorry for the inconvenience!

Currently, we encourage you to use the command-line version of QUAST (see the download and run instructions in the manual). Also, note that you can try to start the QUAST web interface locally since it is also an open-source project available on GitHub.
