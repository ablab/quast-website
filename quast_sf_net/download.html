Download

The current stable release is v4.0. The package includes everything needed for running QUAST, MetaQUAST and Icarus.

QUAST releases are distributed on SourceForge.

You can also download the latest version and extract it from command line: 

    wget https://downloads.sourceforge.net/project/quast/quast-4.0.tar.gz
    tar -xzf quast-4.0.tar.gz
    cd quast-4.0


QUAST releases are also mirrored at GitHub (starting from v4.0).
